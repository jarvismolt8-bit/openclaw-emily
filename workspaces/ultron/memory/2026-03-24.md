## Technical Challenges Encountered
1. **Selector brittleness** - Dynamic modals and div-based buttons
2. **Visibility timing** - Elements present but not visible due to overlays
3. **Form validation** - Required fields hidden until modal opens
4. **Click interception** - Elements covered by other elements
5. **Connection stability** - Multiple websockify instances caused conflicts

## Solutions Implemented
- Multi-strategy click (text → selector → JS injection)
- scrollIntoViewIfNeeded before clicks
- Long wait times for modal animations (2-3 seconds)
- JS fallback with direct event dispatch
- Duplicate process cleanup (killed extra websockify)
