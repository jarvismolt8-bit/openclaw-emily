## Retain
- W @Example: This is a world fact about a system or configuration
- B @Person: This is a biographical fact about someone
- O(c=0.85) @Person: This is an opinion/preference with confidence level
- S: This is a summary or observation

## Format Guide
W = World fact (objective, about systems/configurations)
B = Biographical (facts about people/agents)
O(c=X) = Opinion with confidence 0.0-1.0
S = Summary/Observation
